📊 IT Support Team Performance Dashboard
📌 Project Overview

This project presents an interactive Power BI dashboard that analyzes the performance of an IT support team using customer support ticket data. The dashboard helps in understanding ticket volume, resolution efficiency, customer satisfaction, and product-related issues.

🎯 Problem Statement

Organizations receive a large number of support tickets through different channels. It becomes difficult to:

Track ticket resolution performance
Identify problematic products
Understand customer satisfaction
Analyze channel efficiency

This dashboard aims to provide data-driven insights to improve support operations.

📁 Dataset Description

The dataset contains information related to customer support tickets, including:

Customer details (Age, Gender)
Product Purchased
Ticket ID
Ticket Status (Closed / Pending)
Ticket Priority
Ticket Channel (Email, Phone, Chat, Social Media)
First Response Time
Time to Resolution
Customer Satisfaction Rating
🔍 Exploratory Data Analysis (EDA)

The following steps were performed:

Data cleaning and handling missing values
Data type transformation (time columns adjusted)
Removal of errors and null values
Basic statistical analysis (average, count, distribution)
Identification of key variables for visualization

📈 Key Insights
The system handles a high volume of tickets (5650 total tickets).
Nearly half of the tickets are resolved, indicating scope for improvement.
All communication channels are used almost equally by customers.
Certain products generate more support tickets, indicating recurring issues.
Customer satisfaction is moderate and varies with service quality.
Some channels show higher pending tickets, indicating inefficiencies.
