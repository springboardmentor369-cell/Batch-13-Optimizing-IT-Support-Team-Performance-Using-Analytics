### 📂 Processed Data

**Description:**  
The `Processed Data` folder contains the cleaned and transformed version of the dataset after applying data preprocessing and Exploratory Data Analysis (EDA) steps.

**Transformations Applied:**
- Data cleaning and formatting
- Removal of extra spaces and inconsistencies
- Data type corrections
- Validation of records
- Preparation for visualization and analysis in Power BI

**Purpose:**
- Provide analysis-ready data
- Improve data quality and consistency
- Support accurate dashboard reporting and insights generation

### 🔄 Data Workflow
Raw Data → Data Cleaning & Transformation → Processed Data → Dashboard & Insights
