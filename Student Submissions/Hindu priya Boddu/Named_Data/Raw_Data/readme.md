### 📂 Raw Data

**Description:**  
The `Raw Data` folder contains the original dataset collected from the source before applying any modifications or transformations.

**Purpose:**
- Preserve the original data in its initial format.
- Serve as a backup reference for validation and auditing.
- Ensure transparency in the data analysis process.

**Characteristics:**
- Unmodified dataset
- May contain inconsistencies or formatting issues
- Used as the starting point of the ETL process
