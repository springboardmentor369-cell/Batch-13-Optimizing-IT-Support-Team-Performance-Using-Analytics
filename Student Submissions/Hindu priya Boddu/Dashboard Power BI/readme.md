# 📊 Service Desk Using Data Analysis

## 📌 Overview
This project presents a Power BI dashboard built to analyze student performance data.  
The dashboard helps in understanding how different factors such as gender, ethnicity, parental education, and test preparation impact student scores.

## 🎯 Objective
The main objective of this project is to:
- Analyze student performance across subjects
- Identify key factors influencing academic results
- Provide insights for improving student outcomes

## 📂 Dataset Description
The dataset contains records of 100 students with the following fields:
- Student ID – Unique identifier for each student  
- Gender – Male / Female  
- Ethnicity – Group A, B, C, D, E  
- Parental Education – Education level of parents  
- Test Preparation – Completed / Not Completed  
- Math Score – Marks in mathematics  
- Reading Score – Marks in reading  
- Writing Score – Marks in writing  

## 📊 Key Features of Dashboard
### 🔹 KPI Cards
- Total Students  
- Average Math Score  
- Average Reading Score  
- Average Writing Score  
- Overall Average Score  

### 🔹 Visualizations
- Performance by Gender  
- Performance by Ethnicity  
- Impact of Parental Education  
- Test Preparation Analysis  
- Top Performing Students  

### 🔹 Filters (Slicers)
- Gender  
- Ethnicity  
- Parental Education  
- Test Preparation  

## 📈 Key Insights
- Students who completed test preparation perform better  
- Parental education has a positive impact on student scores  
- Female students slightly outperform male students  
- Performance variation across ethnic groups is minimal  

## 🛠 Tools Used
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Visualization Techniques  

## 🚀 How to Use
1. Open the Power BI file (.pbix)  
2. Use slicers to filter data  
3. Explore visuals to understand insights  
4. Analyze performance based on different factors  

## 📌 Conclusion
This dashboard provides a clear and interactive way to analyze student performance.  
It helps in identifying key areas for improvement and supports data-driven decision-making in education.
