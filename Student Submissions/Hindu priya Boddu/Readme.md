# 🎓 Student Performance Analysis Dashboard


## 📌 Project Overview

The **Student Performance Analysis Dashboard** is a data analytics project developed during the **Infosys Springboard Data Analytics Internship**.

The objective of this project is to analyze student academic performance using data analytics techniques and visualize meaningful insights through an interactive **Power BI dashboard**.

This project demonstrates how raw educational data can be transformed into actionable insights that help educators, institutions, and decision-makers understand factors influencing student performance.

## 🎯 Project Objectives

-  Analyze student performance across subjects  
-  Identify factors affecting academic scores  
-  Perform data cleaning and transformation using Power Query  
-  Apply Exploratory Data Analysis (EDA) concepts  
-  Build interactive dashboards using Power BI  
-  Present insights using data storytelling techniques  


## 🗂️ Dataset Information

The dataset contains performance records of **100 students**.

| Column Name | Description |
|-------------|-------------|
| Student ID | Unique identifier for each student |
| Gender | Student gender |
| Ethnicity | Ethnic group classification (Group A, B, C, D, E) |
| Parental Education | Associate's Degree, Bachelor's Degree, High School, Master's Degree, Some College, Some High School |
| Test Preparation | Completion status of preparation course (Completed / Not Completed) |
| Math Score | Mathematics exam score |
| Reading Score | Reading exam score |
| Writing Score | Writing exam score |

## 🔄 Data Analytics Workflow
Raw Data → Data Cleaning → Data Transformation → EDA → Data Modeling → Visualization → Insights


### 1️⃣ Data Cleaning (Power Query)

- Removed blank rows
- Trimmed extra spaces
- Cleaned non-printable characters
- Standardized categorical values
- Split columns where necessary

### 2️⃣ Data Transformation

- Data type corrections
- Column renaming
- Created calculated fields
- Prepared dataset for analysis

### 3️⃣ Exploratory Data Analysis (EDA)

- Numerical vs categorical analysis
- Distribution of scores
- Outlier identification
- Column quality validation

## 🔍 Key Insights

-  Students who completed test preparation courses showed higher average scores across all subjects.
-  Reading and Writing scores are strongly correlated, indicating language skills consistency.
-  Students with higher parental education levels generally performed better academically.
-  Math scores showed greater variation compared to reading and writing scores.
-  Gender-based differences were minimal but slightly favored reading performance in one group.

## 💡 Recommendations

-  Encourage participation in test preparation programs
-  Provide additional math support sessions
-  Use data-driven monitoring for student progress
-  Design personalized learning strategies
-  Focus early intervention on low-performing groups
