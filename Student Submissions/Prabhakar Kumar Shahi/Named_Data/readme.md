## 📂 Dataset Folder Structure

This folder contains the datasets used in the project.

### It is divided into two sections:
1. **Raw Data**
2. **Processed Data**

---

## 1️⃣ Raw Data

The **Raw Data** folder contains the original dataset collected for this project.

> ⚠️ **Note:**  
> The complete raw dataset contains approximately **2,00,000 rows**, and its file size is **more than 25 MB**.  
> Because GitHub does not allow direct browser upload for files larger than **25 MB**, the full dataset could not be uploaded here.

### 📌 What is available in this repository?
- A **sample dataset of 1,500 rows** is uploaded in the **Raw Data** folder for demonstration and project understanding.

### 📥 Full Raw Dataset (2 Lakh Rows)
The complete raw dataset is available on Google Drive:
