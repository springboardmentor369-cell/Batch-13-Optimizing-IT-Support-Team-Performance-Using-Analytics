# Week 7 - Dashboard Development & Data Visualization  

## Overview  
This week focused on building an interactive dashboard using Power BI and applying data visualization techniques to derive insights from IT support data. The sessions emphasized creating meaningful KPIs, using DAX for calculations, and designing user-friendly visuals for better decision-making.


## Day-wise Learning Summary  

### Monday to Thursday: Dashboard Creation & Visualization  

- Created calculated measures using DAX for KPIs such as:
  - Resolution Rate  
  - Tickets Resolved  
  - High Priority Ticket Count  

- Learned how to use DAX functions like:
  - CALCULATE  
  - COUNTROWS  
  - FILTER  
  - DIVIDE  

- Applied slicers to enable dynamic filtering of data based on:
  - Priority  
  - Ticket Type  
  - Date  

- Built various charts to represent data effectively:
  - Bar charts for department-wise ticket volume  
  - Pie/Donut charts for priority distribution  
  - Line charts for ticket trends over time  
  - Heatmap for weekly and daily ticket analysis  

- Used filters to refine data and improve analysis  

- Designed KPI cards to highlight key metrics clearly  

- Improved dashboard design using:
  - Consistent color themes (Color Hunt)  
  - Proper alignment and spacing  
  - Clear and insight-driven titles  

- Focused on making the dashboard visually appealing and easy to understand  


### Friday & Saturday: Presentation  

- Presented individual dashboards  
- Explained KPIs, visuals, and insights derived from the data  
- Answered questions related to dashboard design and analysis  
- Observed and learned from others' dashboard presentations  
- Improved communication and presentation skills  


## Key Outcomes  

- Gained hands-on experience in building dashboards using Power BI  
- Learned how to write and apply DAX measures for business KPIs  
- Understood how to use slicers and filters for interactive analysis  
- Developed skills in selecting appropriate charts for different data types  
- Improved dashboard design using color and layout principles  
- Built confidence in presenting data insights clearly
