# Week 1 - Introduction & Dataset Selection

## Overview
The first week of the internship focused on building a strong foundation in data concepts and understanding the overall project workflow. The sessions introduced key analytical concepts and tools required for the internship.

## Day-wise Learning Summary

### Day 1: Types of Data
- Introduction to different types of data
- Understanding structured and unstructured data
- Importance of data types in analysis

### Day 2: Introduction to Power Query
- Overview of Power Query
- Understanding how Power Query is used for data cleaning and transformation
- Basic concepts and use cases of Power Query

### Day 3 & Day 4: Dataset Selection
- Given a task to select a dataset based on the assigned domain
- Identified a suitable dataset relevant to IT support operations
- Downloaded the dataset for further analysis
- Installed Microsoft Power BI Desktop for upcoming dashboard development

### Day 5: Query Session & Presentation
- Query session conducted to clarify doubts
- Team members presented the datasets they selected
- Explained dataset structure, domain relevance, and key attributes


## Key Outcomes
- Gained clarity on fundamental data concepts
- Understood the role of Power Query in data cleaning
- Successfully selected a domain-specific dataset
- Prepared the system environment for future analysis and visualization tasks
