

# IT Support Performance Analysis Dashboard

## 📌 Project Title

**Optimizing IT Support Team Performance Using Analytics**

---

## 📖 Project Overview

This project focuses on analyzing IT support ticket data to evaluate and improve the performance of an IT support team. Using Power BI, the project transforms raw data into meaningful insights through interactive dashboards.

The goal is to identify inefficiencies, monitor team performance, and improve overall service quality using data-driven decisions.

---

## 🎯 Problem Statement

IT support teams handle a large number of tickets daily, making it difficult to track performance and identify inefficiencies.

This project aims to:

* Analyze ticket handling performance
* Identify delays and bottlenecks
* Evaluate technician performance
* Improve response and resolution time

---

## 📂 Dataset Description

The dataset contains IT support ticket records, where each row represents a single support request.

### Key Fields:

* Ticket ID
* Category (Hardware, Software, Network, etc.)
* Priority (Low, Medium, High, Critical)
* Assigned Agent
* Ticket Status
* Created Date and Resolved Date
* Resolution Time
* Customer Satisfaction Score

### Dataset Types:

* **Raw Data** → Original dataset
* **Processed Data** → Cleaned and transformed dataset

---

## 🔄 Data Analysis Process

### 1. Data Cleaning

* Removed duplicates and blank rows
* Handled missing values
* Standardized inconsistent values
* Fixed incorrect data types

### 2. Data Transformation

* Split and merged columns
* Cleaned text using Trim and Clean
* Converted text to numeric values
* Formatted date and percentage values

### 3. Exploratory Data Analysis (EDA)

* Identified data types (categorical, numeric, time series)
* Detected outliers
* Analyzed relationships between variables
* Used data profiling tools

### 4. Data Modeling

* Created relationships between tables
* Implemented star schema
* Defined fact and dimension tables

### 5. Measures and KPIs

* Created measures using DAX
* Defined key performance indicators such as:

  * Total Tickets
  * Average Resolution Time
  * Closed Tickets
  * Customer Satisfaction Score

---

## 📊 Dashboard Description

The Power BI dashboard provides an interactive view of IT support performance.

### Features:

* KPI cards for quick insights
* Ticket trends over time
* Technician performance analysis
* Issue category distribution
* Priority distribution
* Interactive filters (Agent, Category, Date, Priority)

---

## ❓ Key Questions Answered

* How many tickets are created, resolved, and pending?
* What is the average resolution time?
* Which agent resolves the most tickets?
* Which issue category occurs most frequently?
* Which priority level requires more attention?
* Are there delays or bottlenecks in the system?

---

## 🔍 Insights

* Certain categories generate more tickets
* Some agents handle higher workloads
* High-priority tickets require faster resolution
* Ticket trends vary over time

---

## 💡 Recommendations

* Balance workload among support agents
* Improve response time for high-priority tickets
* Focus on frequently occurring issue categories
* Use dashboards for continuous monitoring
